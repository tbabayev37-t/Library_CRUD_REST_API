# Library CRUD REST API

Bu layihə kitabxana idarəetmə sistemini simulyasiya edən bir CRUD REST API layihəsidir.

## Checkpoint 1 - Layihə Qurulumu və Entity Dizaynı
- `Author`, `Book` və `Member` modelləri yaradılmışdır.
- Modellər arasındakı əlaqələr Fluent API (`IEntityTypeConfiguration`) vasitəsilə tənzimlənmişdir.
- Entity Framework Core Migrations istifadə edilərək verilənlər bazası formalaşdırılmışdır.

## Texnologiyalar
- .NET 10.0 (C#)
- Entity Framework Core
- MSSQL (SQL Server)
